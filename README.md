# 🚀 Projeto Katherine Johnson - Mentora e educadora Financeira 

Este repositório é um fork do projeto 🤖 Agente Financeiro Inteligente com IA Generativa, desenvolvido como projeto final do Bootcamp Bradesco - GenAI & Dados - Plataforma da DIO.

## 📝 Descrição
Agente Financeiro de IA que utiliza a **Groq API** para guiar jovens profissionais a alcançarem suas metas e monitorar a saúde financeira. O sistema identifica desvios orçamentários e educa sobre educação financeira.

## 🛠️ Tecnologias
* **Python** (Lógica e Integração)
* **Streamlit** (Interface de Usuário)
* **Groq API / Llama 3.1** (Inteligência Artificial)
* **Pandas** (Análise de Dados)

## ⚙️ Como Rodar o Projeto
1. Clone o repositório.
2. Instale as dependências: `pip install -r requirements.txt`.
3. Crie um arquivo `.env` com sua `GROQ_API_KEY`.
4. Execute: `streamlit run src/app.py`.
